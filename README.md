# 📊 YouTube Trend Decay Analyser

## 🎯 Problem Statement
Most YouTube analytics ask "what is trending?"
I asked a smarter question: **"How FAST do trending videos lose popularity?"**

## 🔧 Tech Stack
- Python (Pandas, NumPy, SciPy)
- YouTube Data API v3
- Google Colab
- Looker Studio Dashboard

## 📈 Key Findings
- Average Trend Half-Life = **3.02 hours**
- Fastest decaying video = **0.16 hours**
- Slowest decaying video = **23 hours**
- Music has the highest views overall
- People & Blogs decays fastest on average
